# Obara Ryo Fansite

This is an unofficial fan-made website for artist Obara Ryo.

## Overview

An interactive WebGL-based particle system that forms the artist logo and transitions into a live countdown display.

## Features

- Particle-based logo rendering from image sampling
- Fluid-like interactive motion
- Smooth transition between logo and countdown text
- Real-time day countdown (JST)

## Tech Stack

- WebGL (custom shader)
- JavaScript (Vanilla / modular structure)
- GLSL (particle rendering)
